# eagle-bot-api

## Install
```javascript
git clone https://github.com/lucagump/eagle-cms.git
npm install
```

## Config

After the repository is cloned and modules installed, just place you .env file with tokens and variable you are going to use. 

## Run

```javascript
npm run dev
```
